# Tuning Results

This bundle-local folder contains the tuning outputs retained by the compact review bundle.

## Included Files

- `keep_ratio_dense_scan.json`
- `topn_budget_grid.json`

## Practical Rule

Use these files when the task is about the latest selected operating point or the bounded
`topn x budget` search region discussed in the manuscript.

This compact bundle does not include the full historical sweep, smoke, or dev-split tuning files
from the main repository.
