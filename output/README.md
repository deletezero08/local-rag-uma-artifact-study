# Output

This folder stores generated export material and temporary handoff payloads.

Examples include:

- `chat_context/`
- `doc/`
- `thesis_data/`

Treat this folder as derived output, not as the primary source of truth for experiments or manuscript claims.

For `thesis_data/`, the workspace now keeps only the latest snapshot directory and uses `latest_snapshot.json` as the stable handoff entrypoint.
