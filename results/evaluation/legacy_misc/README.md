# Legacy Misc Results

This folder stores older paper-side support artifacts that are still useful for traceability, but are no longer the default current evidence path.

Examples:

- `ablation_summary.json`
- `formal_intent_verification.json`
- `formal_intent_verification_v2.json`
- `referential_intent_audit.json`

Legacy references must now point directly to `legacy_misc/`.
