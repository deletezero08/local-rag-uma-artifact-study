# Support Runtime Results

This folder stores evaluation outputs that are still useful for engineering-side runtime analysis, but are not the current main paper-track claim files.

Examples:

- `model_comparison.json`
- `niah_report.json`
- `v2_performance_final.json`
- `v2_bench_comparison.json`
- `stress_test_report.json`
- `stress_test_report_v2.json`
- `turboquant_comparison.json`
- `compression_comparison.json`
- `tradeoff_curve.json`

References should now point directly to `support_runtime/`.
