# MemoraRAG Community Code of Conduct

## 我们的宗旨

MemoraRAG 致力于打造一个开放、协作、具有极客精神的本地知识库社区。我们鼓励每一位参与者以理性的态度探寻技术边界，以尊重的态度对待每一位贡献者。

## 期望的行为

- **专业性**: 在 Issue 和 PR 中使用清晰、客观的技术语言。
- **建设性**: 对他人的代码进行 Review 时，重点在于“如何改进”而非简单的“这不对”。
- **包容性**: 欢迎不同技术水平的开发者，耐心解答入门级问题。
- **归属感**: 维护社区声誉，保护项目代码的纯净性。

## 不可接受的行为

- **排他性**: 任何基于地域、性别、身份、技术栈取向的歧视或人身攻击。
- **恶意性**: 故意提交含有恶意代码、后门或可能导致用户隐私泄露的 PR。
- **骚扰性**: 在讨论区进行无关的争吵、辱骂或持续性的负面骚扰。

## 责任与执行

项目维护团队负责解释并执行本准则。对于违反准则的行为，维护者有权根据严重程度采取以下措施：
1. **私人警告**：提醒并纠正其不当言论。
2. **内容删除**：直接删除或锁定争议 Issue/PR。
3. **临时/永久封禁**：限制其参与项目开发的权限。
